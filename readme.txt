A simple, free Wordpress Newsletter Generator compatible with Sendy.co.

== Description ==

WP Sendy Newsletter Generator is a simple no frills plugin that creates a simple and clean newsletter template that is mobile friendly and automatically compatible with Sendy.co.

Not familiar with Sendy.co? Check out **[What Is Sendy.co](https://www.wpsendynewsletter.com)**

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the Sendy Newsletter options screen to configure the plugin.
